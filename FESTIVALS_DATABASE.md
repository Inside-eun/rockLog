# 국내 주요 페스티벌 데이터베이스

2026년 4월 27일 기준 이전 페스티벌 목록

## 대형 페스티벌

### 1. 지산 밸리 록 페스티벌
- **시기**: 7월 말 (3일간)
- **장소**: 경기도 이천시 지산포레스트리조트
- **장르**: 록/메탈
- **인스타그램**: @jisanfestival
- **홈페이지**: https://www.valleyrockfestival.com
- **특징**: 국내 최대 규모 록 페스티벌
- **라인업**: 해외 헤드라이너 + 국내 록밴드

### 2. 펜타포트 록 페스티벌
- **시기**: 8월 초~중순 (3일간)
- **장소**: 인천 송도 달빛축제공원
- **장르**: 록/인디
- **인스타그램**: @pentaport_rockfestival
- **홈페이지**: https://www.pentaportrock.com
- **특징**: 인천 대표 페스티벌
- **라인업**: 얼터너티브 록, 인디록 중심

### 3. 그랜드 민트 페스티벌
- **시기**: 10월 중순 (2일간)
- **장소**: 서울 올림픽공원
- **장르**: 인디/어쿠스틱
- **인스타그램**: @grandmintfestival
- **홈페이지**: https://www.grandmintfestival.com
- **특징**: 감성 인디 페스티벌
- **라인업**: 국내 인디 뮤지션 중심

### 4. 서울재즈페스티벌
- **시기**: 5월 말 (3일간)
- **장소**: 서울 올림픽공원
- **장르**: 재즈
- **인스타그램**: @seouljazzfestival
- **홈페이지**: https://www.seouljazz.co.kr
- **특징**: 국내 최대 재즈 페스티벌
- **라인업**: 세계 정상급 재즈 뮤지션

### 5. 현대카드 슈퍼콘서트 시리즈
- **시기**: 연중 (부정기)
- **장소**: 서울 잠실종합운동장, 고척스카이돔 등
- **장르**: 팝/록
- **인스타그램**: @hyundaicard
- **특징**: 세계적 아티스트 단독 공연
- **예시**: Coldplay, Bruno Mars, The Weeknd

## 데이터 수집 방법

### 1단계: 공식 채널 확인
- 페스티벌 공식 인스타그램
- 공식 홈페이지
- 공식 페이스북

### 2단계: 타임테이블 찾기
- 인스타그램 게시물 검색 (해시태그: #타임테이블 #timetable)
- 공식 홈페이지 공지사항
- 페이스북 이벤트 페이지

### 3단계: 데이터 입력
- 관리자 페이지에서 페스티벌 추가
- CSV로 타임테이블 일괄 업로드
- 검증 후 저장
