# 페스티벌 데이터 스크래핑 가이드

## 개요

이 프로젝트는 Puppeteer를 사용하여 국내 페스티벌 정보를 자동으로 수집합니다.

## 설치

```bash
npm install puppeteer
```

## 사용 방법

### 1. 커맨드라인에서 실행

```bash
npm run scrape
```

### 2. API를 통한 실행

```bash
curl -X POST http://localhost:3000/api/scrape
```

## 스크래핑 대상 사이트

### 구현됨
- ✅ 인터파크 티켓 (기본 구조)

### 구현 예정
- ⏳ 멜론티켓
- ⏳ 예스24 티켓
- ⏳ 개별 페스티벌 공식 사이트

## 주의사항

1. **웹사이트 이용약관 확인**: 스크래핑 전에 각 사이트의 이용약관을 확인하세요.

2. **Rate Limiting**: 과도한 요청으로 서버에 부하를 주지 않도록 주의하세요.

3. **HTML 구조 변경**: 웹사이트 구조가 변경되면 스크래핑 로직도 업데이트가 필요합니다.

4. **법적 책임**: 스크래핑한 데이터의 사용은 사용자의 책임입니다.

## 데이터 구조

### Festival
- name: 페스티벌 이름
- startDate: 시작 날짜
- endDate: 종료 날짜
- location: 개최 장소
- genre: 장르
- imageUrl: 포스터 이미지 URL
- description: 설명

### Performance
- artistName: 아티스트 이름
- startTime: 공연 시작 시간
- endTime: 공연 종료 시간
- stage: 스테이지 이름
- durationMinutes: 공연 시간 (분)
